# mbuster_to_fireflyimport
Konvertiert Exportdateien der App MoneyBuster in ein importfertiges CSV-Format (inkl. optionaler config.json) für den Firefly III Data Importer. Übernimmt Datums- und Betragsformatierung, Vorzeichen sowie das Feld-Mapping zwischen beiden Tools, damit Transaktionen ohne manuelle Nacharbeit in Firefly III importiert werden können.
